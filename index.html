Workflow Process

Data Analysis is the collection, transformation, and organization of data in order to draw conclusions, make predictions and carry out analysis-driven decision-making.
Every analyst has a preferred process of work, this is mine.

1. ASK

Define the Problem statement
Understand the Stakeholders expectations and desires
Use Structured thinking and ask effective questions
Communicate with others

2. PREPARE

Collect, Organize and Protect data
Maintain Data Integrity
Define data on basis of bias and cleanliness

3. PROCESS

Create and transform data for upcoming analysis process
Test and Clean data that cause any anomalies
Verify and Report on cleaning results

4. ANALYZE

Analyze data using tools to draw useful conclusions
Make predictions and recomendations
Make data-driven decisions

5. SHARE

Interpret results and share them with others to help stakeholders make effective decisions
Bring data to life
Communicate to help others understand results

6. ACT

Apply insights to work in order to solve the orignal problem
Make Decisions
